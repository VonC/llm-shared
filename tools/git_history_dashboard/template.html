Observations

  • The first year (June 2024 to mid-2025) is almost completely empty — the project barely moved, then activity exploded around the week of 8 Sep 2025 and has run flat-out ever since, straight through to June 2026.
  • feat leads the all-time mix (764 commits), with tests and refactors tracking each other close behind (584 vs 472) — the _tdd.py discipline shows up in the data. docs (625) swells through the planning-heavy v9.5.0–v9.8.0 polluted-PDF and resources-isolation work, which is dense with plan / design / proposal markdown.
  • Hour 21:00 is the single biggest commit hour (327), with a clear evening peak from 20:00–23:00. Mornings 8:00–11:00 are the secondary push. The 04:00–06:00 stretch is essentially dead.
  • Thursday edges out Tuesday as the most productive weekday (545 vs 537), but the spread is remarkably flat — Sunday (498) outpaces three weekdays. This is a 7-day-a-week project.
  • Top scopes are led by domain, web, plan, workflow and release — the hexagonal/ports-and-adapters split (domain, adapter, bootstrap) is visible in scope frequency, alongside the feature threads (sentinel, polluted-pdf) and a steady drumbeat of point releases.
